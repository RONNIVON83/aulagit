Explorando os Fundamentos da Web

È uma jornada empolgante pelo mundo da criação e desenvolvimento de páginas da Web. Nesta disciplina, os alunos terão a oportunidadeúnica de mergulhar profundamente nos conceitos essenciais que sustentam a construção da internet moderna. Combinando teoria e prática, esta disciplina é projetada para fornecer uma base sólida para compreender e dominar os elementos cruciais que constituem a espinha dorsal da Web.

Objetivos de aprendizagem

HTML(Hypertext Markup Language):Os alunos aprenderão a estrutura básica da Web usando HTML, a marcação de textos, imagens, links e listas, entendendo como criar páginas Web acessíveis e bem organizadas.

CSS(Cascading Style sheets: os alunos descobrirão como dar vida ás páginas usando CSS.Aprendendo a aplicar estilos, cores, fontese layout para tornar o conteúdo visualmente atraente e responsivo em diferentes dispositivos.
